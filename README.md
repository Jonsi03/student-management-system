🎓 Student Management System

A simple Student Management System developed using Python and Object-Oriented Programming (OOP). This project helps manage student records with CSV file handling and exception handling.

🚀 Features

- Add Student
- View All Students
- Search Student by ID
- Search Student by Name
- Update Student Details
- Delete Student
- Save Data to CSV File
- Load Data from CSV File
- Duplicate ID Validation
- Exception Handling

🛠️ Technologies Used

- Python
- Object-Oriented Programming (OOP)
- CSV Module
- File Handling
- Exception Handling

▶️ How to Run

1. Download or Clone this repository.
2. Open the project in VS Code.
3. Run the following command:

```bash
python student_management_system.py
```
